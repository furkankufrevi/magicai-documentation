Installation

This section gives informations about how to install MagicAI.

File Structure

  • CSS Files: /../public/assets/css
  • Tailwind CSS Files: /../resources/css
  • JavaScript Files: /../public/assets/js
  • Frontend Assets: /../public/assets/frontend
  • Fonts, images, components: /../public/assets/

Install Laravel

This script uses Laravel Framework for backend. All files are set, and you don't have to install or update anything.
Want to know more about laravel, click for Laravel documentation

Server Requirements

  • PHP >= 8.1
  • Ctype PHP Extension
  • cURL PHP Extension
  • DOM PHP Extension
  • Fileinfo PHP Extension
  • Filter PHP Extension
  • Hash PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PCRE PHP Extension
  • PDO PHP Extension
  • Session PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

If You Are Installing MagicAI to Shared Hosting

Change your document root to public folder of the magicAI. /httpdocs/public or public_html/public. See how to? CPanel, Plesk.

Create A Database

Create a database with utf8mb4_general_ci character encoding and give all permissions. Note your database name, user name and password.

Environment (.env) File

There is an environment (.env) file at the root directory of the project. Open it with your IDE or text editor.

Warning

Make sure your IDE or your text editor uses UTF-8 character encoding.

It should look like this;


APP_NAME=Laravel
APP_ENV=production
APP_KEY=base64:FNDGJM5KJ0nx9yhWVwu+EkJjMa+OXRESlbEQXPjO/FM=
APP_DEBUG=false
APP_URL=http://YOUR_SITE_URL
LOG_CHANNEL=stack
LOG_DEPRECATIONS_CHANNEL=null
LOG_LEVEL=debug
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=magicai
DB_USERNAME=root
DB_PASSWORD=password

BROADCAST_DRIVER=log
CACHE_DRIVER=file
FILESYSTEM_DISK=local
QUEUE_CONNECTION=sync
SESSION_DRIVER=file
SESSION_LIFETIME=120

MEMCACHED_HOST=127.0.0.1

REDIS_HOST=127.0.0.1
REDIS_PASSWORD=null
REDIS_PORT=6379

MAIL_MAILER=smtp
MAIL_HOST=mailpit
MAIL_PORT=1025
MAIL_USERNAME=null
MAIL_PASSWORD=null
MAIL_ENCRYPTION=null
MAIL_FROM_ADDRESS="hello@example.com"
MAIL_FROM_NAME="${APP_NAME}"

AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_DEFAULT_REGION=us-east-1
AWS_BUCKET=
AWS_USE_PATH_STYLE_ENDPOINT=false


VITE_PUSHER_APP_KEY="${PUSHER_APP_KEY}"
VITE_PUSHER_HOST="${PUSHER_HOST}"
VITE_PUSHER_PORT="${PUSHER_PORT}"
VITE_PUSHER_SCHEME="${PUSHER_SCHEME}"
VITE_PUSHER_APP_CLUSTER="${PUSHER_APP_CLUSTER}"
Change fields with:
  • APP_URL Your website's full url. For example; https://liquid-themes.com
  • DB_DATABASE: Your database table's name.
  • DB_USERNAME: Your database table's user name.
  • DB_PASSWORD: Your database table's password.

And save the .env file
