Installation

This section gives informations about how to install MagicAI.

File Structure

  • CSS Files: /../public/assets/css
  • Tailwind CSS Files: /../resources/css
  • JavaScript Files: /../public/assets/js
  • Frontend Assets: /../public/assets/frontend
  • Fonts, images, components: /../public/assets/

Server Requirements

This script uses Laravel Framework for backend. All files are set, and you don't have to install or update anything.
Want to know more about laravel, click for Laravel documentation

Server Requirements

  • PHP >= 8.1
  • MariaDB >= 10.5 or(and) Mysql => 5.7.X
  • Ctype PHP Extension
  • cURL PHP Extension
  • DOM PHP Extension
  • Fileinfo PHP Extension
  • Filter PHP Extension
  • Hash PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PCRE PHP Extension
  • PDO PHP Extension
  • Session PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

It is recommended to increase PHP's max_execution_time.

See how to? CPanel, Plesk, Others (Linux Servers).

If You Are Installing MagicAI to Shared Hosting

Change your document root to public folder of the magicAI. /httpdocs/public or public_html/public. See how to? CPanel, Plesk.

Create A Database

Create a database with utf8mb4_general_ci character encoding and give all permissions. Note your database name, user name and password.

How to create database?

Plesk, CPanel, MacOS, Windows. If this links are not clear please search for "how to install mysql and phpmyadmin to YOUR_OS_NAME?".

Install Script

Tip

It is important to enter correct database information to the form.

Warning

Make sure your database connection has no problems.

To install the script;

  1. Go to: https://yourdomainname.com/install-script
  2. Fill the form
  3. Click to install button once.
  4. The installer will install the script.

The installer will do rest of work and log you in to the panel.

Login Information:
Email: admin@admin.com
Password: 12345678

After the installation fill the informations on the settings pages.

Install Node.js (Optional)

If you are planning to make changes on assets, please install node.js to compile your changes.

Install node.js to your server or your localhost.

Upgrade

Upgrading is easy with magicAI.

Upgrade Files

Check your version from the footer of the dashboard. Compare the version number with the release.

Tip

All future releases contains all previous updates so you don't have to install them seperated.

Download the compressed file and unzip the file in the project root directory.

Danger

If you make adjustments with project assets, routing, functions and all other things they all will reset after the update. It is recommended to do all changes with additional files.

Install Update

To update the script just go to https://yourdomainname.com/upgrade-script

The system will do the update and show you a message if further instructions are needed.
